Grocery Delivery System

Developer: Rebekah Tolliver

Description:
The Grocery Delivery System is a web-based application developed using HTML, CSS, and JavaScript. 
It allows users to browse grocery items, search and filter products, manage a shopping cart, and 
place delivery orders. The system also simulates real-time delivery status updates.

Key Features:
- User login and registration (local storage-based authentication)
- Product browsing with category filtering
- Real-time search functionality
- Dynamic shopping cart with quantity management
- Automated checkout process
- Delivery status simulation (Preparing → Out for Delivery → Delivered)
- Order history tracking with persistent storage

Data Storage:
This application uses browser Local Storage to simulate a backend database. 
User data, cart items, and order history are stored locally in the browser.

System Requirements:
- Desktop or laptop computer
- Modern web browser (Chrome, Edge, Firefox, or Safari)
- Internet connection

How to Run the Project:
1. Download or clone the project files.
2. Open login.html in a web browser.
3. Register a new account or log in.
4. Use the system to browse products and place orders.

Notes:
- No installation or external dependencies are required.
- This is a front-end simulation and does not connect to a real backend.
